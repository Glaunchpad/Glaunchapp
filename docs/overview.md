# GLaunch Overview

GLaunch is a web application that combines an AI conversational interface with on-chain token launch tooling on the Base network.

## What GLaunch Does

1. **Launch** — Users describe a token in natural language; GLA orchestrates metadata, image upload, and deployment through Flaunch.
2. **Trade & Track** — Market views surface charts, trending tokens, and external DEX links.
3. **Manage** — Wallet pages support balances, withdrawals, royalty claims, and launch history.
4. **Analyze** — PnL tooling helps users understand positions and performance over time.
