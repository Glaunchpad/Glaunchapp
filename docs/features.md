# Features

## Terminal & GLA

| Capability | Description |
|------------|-------------|
| Token deploy | Launch tokens on Base via Flaunch with image and metadata |
| Transfer | Send ETH or USDC to addresses or ENS-like names |
| Claim | Claim earnings and royalties from launches |
| Market scan | Surface trending and relevant market context |
| Token search | Look up tokens by name, symbol, or address |
| Portfolio | Read balances and positions for connected wallet |

## Market

- Featured and trending token listings
- Live price tickers
- Embedded charts (DexScreener, GeckoTerminal, native pool views)
- Quick links to external explorers and DEX pages

## Wallet

- Privy login (Google, X, Farcaster)
- Embedded Base wallet
- Withdraw flows
- Claim buttons for tokens and royalties
- Transaction history context

## My Launches

- Per-user launch dashboard
- Revenue and royalty visibility
- Links back to market and Flaunch profiles

## PnL Analytics

- Position-oriented performance views
- Historical context for launched and held tokens

## Launch Guardrails

- Daily deploy quota per user
- Cooldown period between deploys
- Clear user-facing error messages when limits are hit
