# Contributing

Thank you for your interest in GLaunch!

## What You Can Contribute Here

This public repository accepts:

- Documentation improvements (`docs/`, `README.md`)
- Issue templates and GitHub workflow suggestions
- Bug reports and feature requests
- Changelog entries for user-visible product changes

## What We Cannot Accept Here

- Pull requests containing application source code (the codebase is private)
- API keys, tokens, or credentials of any kind
- Exploits or vulnerability details (see [SECURITY.md](SECURITY.md))

## How to Report Bugs

1. Search [existing issues](https://github.com/Glaunchpad/Glaunch/issues) first
2. Open a [Bug Report](https://github.com/Glaunchpad/Glaunch/issues/new?template=bug_report.yml)
3. Include steps to reproduce, expected vs actual behavior, and screenshots if helpful

## How to Request Features

Open a [Feature Request](https://github.com/Glaunchpad/Glaunch/issues/new?template=feature_request.yml) with a clear problem statement and proposed outcome.

## Documentation PRs

1. Fork the repository
2. Create a branch: `docs/your-topic`
3. Submit a PR with a concise description of changes

## Code of Conduct

Be respectful and constructive. See [CODE_OF_CONDUCT.md](CODE_OF_CONDUCT.md).
