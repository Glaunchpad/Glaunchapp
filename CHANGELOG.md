# Changelog

All notable product changes are documented here. This repo tracks **product-facing** updates; internal implementation details remain private.

The format is based on [Keep a Changelog](https://keepachangelog.com/en/1.1.0/).

## [0.1.0] - 2026-06-03

### Added

- Public repository with product documentation and community templates
- GLA terminal with deploy, transfer, claim, and market tools
- Market page with trending tokens and chart embeds
- Wallet dashboard with Privy authentication
- My Launches (Flaunch) integration
- PnL analytics module
- Launch rate limits and cooldown enforcement

### Infrastructure

- Base mainnet deployment
- PostgreSQL-backed deploy logging
- Integrations: Flaunch, Privy, DexScreener, GeckoTerminal, CoinGecko
